# **App Name**: LocalDink: AI Pickleball Scheduler

## Core Features:

- AI Scheduling Agent (Robin): An AI agent that handles all communication related to scheduling pickleball games - aka Game Sessions.
- Game Session Invitation: Sends game session invites to selected players via SMS or in-app notifications.
- RSVP and Confirmation: Manages RSVPs and confirms players based on their responses, notifying alternates as needed using the AI scheduler tool, Robin.
- Automated Cancellation Management: Handles cancellations and automatically re-notifies alternates to fill the open spot.
- Profile Management: Players can create and manage their profiles, specifying default preferences like doubles, home courts, groups, and availability.
- Player Invitation: A player can easily invite as many other players to a game as they'd like.
- Court Management: The App can manage the choice of the courts, setting a default home court, but can have "favorites" for quick reference.
- Game Sessions Defaults: Game Sessions are default "doubles" but can be Singles or as many players as requested.
- Organizer: Organizer is the player who invites other players from their contact within the app
- Groups: Groups are collections of players for common groups
- Robin can discern names: Robin is a smart AI and can discern names, if only first name is provided.
- Firestore Database: Utilize Firestore for data storage and retrieval.

## Style Guidelines:

- Primary Color: Primary Green (#2B6B21): The deep green from your logo background - perfect for headers, footers, and primary buttons
- Primary Color: Primary Yellow (#FFD600): The bright yellow from the ball and border - ideal for accents and call-to-action elements
- Primary Color: Primary White (#FFFFFF): The clean white from the paddle - excellent for background areas and text on dark colors
- Secondary Color: Secondary Dark Green (#1D4A16): A slightly darker shade of green for hover states and visual hierarchy
- Secondary Color: Secondary Light Yellow (#FFE566): A lighter yellow for backgrounds, hover states, and secondary elements
- Light Gray (#F5F5F5): For content areas and subtle backgrounds
- Text Color: Dark Text (#333333): For main body text on light backgrounds
- Text Color: Light Text (#FFFFFF): For text on dark backgrounds
- Text Color: Accent Text (#1D4A16): For headings and important text elements
- Body and headline font: 'PT Sans', a sans-serif, to provide clarity.
- Use pickleball-themed icons, bold and friendly to illustrate different functions within the app. Example: paddles, balls, and court markings.
- Robin, the AI Agent, is a very clean, simple bird icon
- Clean and intuitive layout optimized for mobile viewing with clear calls to action (CTAs). The app should look good both on iOS and Android devices.
- Subtle animations for user interactions, like confirming a game or when Robin sends updates. Animation style should be friendly and approachable, not distracting.
